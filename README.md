# Beverage Sales Dashboard

## Project Overview
This Power BI dashboard analyzes beverage sales performance, product trends, and business insights using interactive visualizations.

## Tools Used
- Power BI

## Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Product Performance
- Interactive Filters
- Business Insights

## Key Insights
- Sales performance trends identified
- Product category analysis performed
- Business growth insights visualized
